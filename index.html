Hola, buen día. Como práctica sobre los selectores de CSS se solicita lo siguiente: 1. Crear un tablero de ajedrez usando el selector nth-child. 2. El tablero se tiene que crear usando tablas. 3. Usen este servicio http://codepen.io para crear su tablero. 4. Ya que lo tengan listo, favor de pegar la url del tablero. Nota: favor de no enviar archivos, peguen solo la url. Acá los dejo un ejemplo usando div: http://codepen.io/KnightJK/pen/wBpBdd Les dejó el que yo realice usando listas: http://codepen.io/anon/pen/Xbzbre Les dejó un video donde explican cómo hacerlo con tablas: https://www.youtube.com/watch? v=24QjdOuMMPA Saludos.
